Create a new hook `src/hooks/useKeyboardShortcuts.ts` that:
1. Accepts `onEscape` and `onSlash` callback props (both optional functions)
2. Adds a global `keydown` event listener using `useEffect`
3. On "Escape" key: calls `onEscape()` callback (if provided)
4. On "/" key: checks if user is in an input/textarea element - if NOT, calls `onSlash()` callback
5. Checks active element: `const tag = document.activeElement?.tagName?.toLowerCase(); const isTyping = tag === 'input' || tag === 'textarea';`
6. If `isTyping` is true, skips the "/" handler (context-aware per D-04)
7. Returns cleanup function to remove listener on unmount
8. Exports the hook as a named export

Example structure:
```typescript
import { useEffect, useCallback } from 'react';

export function useKeyboardShortcuts({
  onEscape,
  onSlash,
}: {
  onEscape?: () => void;
  onSlash?: () => void;
}) {
  const handleKeyDown = useCallback((event: KeyboardEvent) => {
    // Escape: always works
    if (event.key === 'Escape' && onEscape) {
      onEscape();
    }
    // Slash: only when not typing
    if (event.key === '/' && onSlash) {
      const tag = document.activeElement?.tagName?.toLowerCase();
      const isTyping = tag === 'input' || tag === 'textarea';
      if (!isTyping) {
        event.preventDefault(); // Prevent "/" from being typed
        onSlash();
      }
    }
  }, [onEscape, onSlash]);

  useEffect(() => {
    document.addEventListener('keydown', handleKeyDown);
    return () => document.removeEventListener('keydown', handleKeyDown);
  }, [handleKeyDown]);
}
```
